Getting Started

Simple tutorial on how to get started

This is your first step in the tutorial To get started, make sure you have installed npm. Check it by running: npm -v If that ran successfully, you can continue.
Now simply run: npm i -g @p-lang/plang

Now you can run a special command called pint. This is the P interpreter, aka the program you will run your P programs with.
You can also now use the prepl (P read-eval-proccess loop) to run P directly in your terminal to start the repl just run pint with no arguments to run a single file using pint run:
pint <filepath>

Congrats you have successfully ran pint for the first time ever!
