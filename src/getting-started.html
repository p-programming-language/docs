Getting Started

Simple tutorial on how to get started

Installing

This is your first step in the tutorial To get started, make sure you have installed npm. Check it by running: npm -v
If that ran successfully, you can continue.
Now simply run: npm i -g @p-lang/plang

Running pint for the first time

Now you can run a special command called pint. This is the P interpreter, aka the program you will run your P programs with.

You can also now use the prepl (P read-eval-print loop) to run P directly in your terminal to start the repl just run pint
To run a single file using pint run:
pint <filepath>
Congrats you have successfully ran pint for the first time ever!

Your very first P program

For this tutorial you need a text editor (preferably an IDE like Visual Studio Code)

If you are using Visual Studio Code you can install vscode-p to get features like P syntax highlighting for vscode!

It's time to write your first P program! Just a simple program that prints "Hello, Plang!" on the screen.

Get started by making a hello.p file, now open the file with your preferred IDE or text editor. Now it's time to code!

To get started, we need to import the writeln function from the standard input and output library (std.io). To do that, add this line to the top of the file:

use writeln from @std.io

(We need to import writeln to print a line on the screen using P)

Now writeln is accessible in the program, so now we can add another line to the file:

writeln("Hello, Plang!")

Congrats you have now written your first P program, to run it run:

pint hello.p

This should be the output

$ pint hello.p
Hello, Plang!

The entire code:

use writeln from @std.io
            
writeln("Hello, Plang!")
